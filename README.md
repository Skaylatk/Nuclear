# Como Usar:

## Instalados:
- JDK 17+

## Instalação

- Clone Esse projeto Utilizando o `git clone`
- Encontre a pasta `src` e compile o arquivo `Nuclear.java` usando o javac
- Depois de Compilar, execute o arquivo Usando o Comando `java Nuclear`
